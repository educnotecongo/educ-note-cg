## 🚀 Déploiement rapide

1. Crée un repo GitHub nommé : `educnotecongo.github.io`
2. Copie tous ces fichiers dedans
3. Push vers GitHub
4. Va dans **Settings → Pages** → Active sur branche `main`
5. ✅ Site en ligne sous 2 min !

---

## ⚙️ Formulaire de contact

1. Crée un compte gratuit sur [Formspree](https://formspree.io)
2. Remplace `YOUR_FORM_ID` dans `contact.html` par ton ID Formspree
